# 🚀 SVG to CSS Generator

SVG코드를 CSS로 변환해주는 도구입니다.

## ✨ 주요 기능

- 📝 SVG코드를 CSS코드로 생성
- 🚀 SVG코드를 CSS background-image 속성으로 생성

## 📌 사용 방법

1. 원하는 SVG코드를 복사합니다.
2. SVG코드 삽입부분에 해당 코드를 붙여 넣기 합니다.
3. 변환하여 원하늨 코드를 복사하여 사용합니다.
